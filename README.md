Dobryweb\NetteExt
==================

Add-ons extending functionality, what we missed in [Nette Framework](http://nette.org).


Installation
------------

The best way to install Dobryweb/NetteExt is using  [Composer](http://getcomposer.org/):

```sh
$ composer require dobryweb/nette-ext
```
